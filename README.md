# kontakt-gql

[![Greenkeeper badge](https://badges.greenkeeper.io/shekohex/kontakt-gql.svg)](https://greenkeeper.io/)

## Description

Yet another Contact Manager Using Nestjs and GraphQL

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
